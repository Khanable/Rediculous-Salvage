Hi, Welcome to the 90's. The viewer is here. Only tested with up to date chrome, but any modern browser should work (firefox/edge, not ie11).
viewer
The ship building algorithm is good enough for me to continue with making the rest of the game. I want to improve the forward decision making and might go back and improve other sections. Note: the viewer will store its state across reloading the page but not closing and opening the browser.

Controls
  • c: to centre
  • f: to algin forward
  • o: to toggle origin axes
  • e: to toggle showing thruster keys
  • s: to toggle showing seed input
  • t: to toggle showing stage rendering

Dialog element controls
This is the order of input capture also

Seed input
  • Only numbers accepted
  • backspace to erase one character
  • cursor left/right keys increment/decrement seed by 1
Stage rendering
  • Arrow keys up/down change selection
  • Arrow key right toggles enable/disable
